Introduction

Lobby Server is a general-purpose WebSocket-based lobby server.

The authentication method used is JSON Web Token. The client must send the token in the connection URL and messages.

wss://localhost:9000/token

Errors

The server uses the following error schema:

Field Type Value
event string error
data json
Field Type Description
error number The error code.
message string The error message.
uuid string The message universally unique identifier.
Error Description
100 Generic error.
101 Validation error.
102 Invalid token.
103 User not found.
104 User already exists.
105 User not in room.
106 User already in room.
107 Room not found.
108 Room already exists.
109 Invalid room password.
